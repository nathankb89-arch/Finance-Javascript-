# 💰 Finance+

*A Simple, Smart Personal Finance Tracker*

---

## 📌 Project Overview

**Finance+** is a web-based personal finance tracking application developed as an end-of-module project for a Software Development course. The app is designed to help users manage their income, expenses, and financial goals in a clean, intuitive, and efficient way.

With a focus on simplicity and usability, Finance+ empowers users to take control of their finances by providing clear insights into their spending habits and savings progress.

---

## 🚀 Features

* **Income & Expense Tracking**
  Easily record and categorize your daily transactions.

* **Real-Time Balance Calculation**
  Automatically updates your total balance as transactions are added or removed.

* **Savings Goals**
  Set financial goals and monitor your progress over time.

* **Local Data Persistence**
  Uses browser `localStorage` to save user data, ensuring information remains available between sessions.

* **Responsive Design**
  Clean and modern UI that works across desktop and mobile devices.

* **User-Friendly Interface**
  Minimalist design focused on clarity and ease of use.

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the application
* **CSS3 / Tailwind CSS** – Styling and layout
* **JavaScript (ES6)** – Application logic and interactivity
* **LocalStorage API** – Data persistence

1. Clone the repository:

   ```bash

   ```

2. Navigate into the project folder:

   ```bash

   ```

   * Double-click the file
   * OR use a live server extension (recommended)

## 💡 How It Works

* Users input their income and expenses through the interface.
* JavaScript processes and updates the total balance dynamically.
* All data is stored in the browser using `localStorage`.
* The app retrieves saved data on reload, ensuring continuity.
